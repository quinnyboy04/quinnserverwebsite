Tree Pots

Terug naar Nederlandse hoofdpagina.

 

Tree Pots is een plugin door VoodooBeard. Het is een erg simpele plugin waardoor je hout kan krijgen als je een boom met een pot op een hopper hebt. Hier is een plaatje van hoe het werkt:

 

Benodigdheden:

- Pot

- Sapling (welke je wilt)

- Hopper

- Diamond Axe

- Item Frame

- Opslag (als je wilt, is handig)

 

Elke 10 minuten genereerd de machine een van de volgende dingen:

- Log (liggend aan welke sapling je hebt)

- Leaves (liggend aan welke sapling je hebt)

- Sapling (liggend aan welke sapling je hebt)

- Apple

- Stick
